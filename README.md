# Expense Tracker

A personal expense tracking app built with React.

## Live Demo
https://expense-tracker-83se274yr-dyuthi-hareesh-s-projects.vercel.app

## Features
- Add expenses with description, amount and category
- Filter expenses by category (Food, Transport, Shopping, Bills, Other)
- Visual spending breakdown with category bars
- Tracks highest spend category automatically
- Data persists across sessions using localStorage

## Tech Stack
- React (useState, useEffect hooks)
- CSS (custom, no libraries)
- localStorage for data persistence
- Deployed on Vercel

## How to Run Locally
- git clone https://github.com/DyuthiHareesh/expense-tracker.git
- cd expense-tracker
- npm install
- npm start
